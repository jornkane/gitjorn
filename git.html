Staging:
Når du har valgt git add minfil.py så er den fila staged
Commiting:
Når du velger git commit -m så lagrer vi en save av fila slik den er akkurat nå. 
Husk å ta med beskrivelse også
git commit -m "Dette er første versjon av programmet, kun hello world nå"
