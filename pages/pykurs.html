Lister og løkker

Dette er mye på en gang, men vi setter av tid til å jobbe med emnet 
talliste = ['a','b','c','d'] # lister har alltid []
print("Lista har", len(talliste), "elementer.")
for a in range (len(talliste)): # len er lengden av lista
  print('Element nr',a,'er:',talliste[a])
print('--------------------------------')
# Dette er en kommentar, det er bare en huskelapp til meg selv
# Loopen under legger til 4 tall på lista
for a in range (len(talliste)):
  print('Dette er a:',a)
  talliste.append(a) # legger til bakerst i lista
print(talliste)
print(talliste[5]) # printer element nr 5
While-løkke: 
dividend = int(input("Skriv et tall "))
divisor =2
kvotient = []
print('Dividend=',dividend,',divisor=',divisor)
print('Kvotient:')
while dividend > 0.9:
  dividend = dividend / divisor
  print(dividend)
  kvotient.append(dividend)
print('Liste med kvotientene:',kvotient)

Libraries. Eksempler er random, math, numpy, matplotlib.pyplot og mange fler
Først numpy og tallistevarianten np.array: 
import numpy as np
bakterier = np.zeros(30) # Lager en array med 30 tomme plasser
bakterier[0] = 3 # vi starter med 3 bakterier
for i in range(1, 30):
  bakterier[i] = 2*bakterier[i-1]
print(bakterier,type(bakterier))
